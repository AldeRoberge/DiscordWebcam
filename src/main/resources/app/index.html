DiscordWebcam

Getting Started Guide

This is a quick tutorial that will explain how to set up DiscordWebcam.

1. CREATE A DISCORD SERVER

1. Create a new Discord Account if you don't already have one
2. Create a new Discord Server
3. Activate Discord developper options to find your Discord Channel ID

2. CREATE A DISCORD BOT

DiscordWebcam requires a Discord bot to send notifications. 

1. Create a new Discord application
2. Create a new Bot
3. Copy the Token (not Client ID or Secret)
4. Make your bot join the new server

3. LAUNCHING discordwebcam

Tip: You need Java to use the software. Download Java here.

1. Find and run the DiscordWebcam.jar.
2. A prompt will open showing all available settings. Enter your Discord Bot Token and Discord Channel ID.
3. Close the window.

4. ADDING CAMERAS TO DISCORDWEBCAM

1. Plug a USB camera to the computer
2. Go to File -> Detect local camera(s)
3. The camera should show up. If it does not, try looking at the logger. 

Tip: To open the logger, go to Edit -> Show logger

5. DETECTING MOVEMENT

1. Open the settings of the camera you want to allow movement detection
2. Turn on movement detection and sending to Discord

Tip: Apply "show motion boxes" to see when motion is detected.
